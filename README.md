# JSON Fetch CLI

A simple command-line interface (CLI) tool for accessing the JSONPlaceholder API. JSON Fetch CLI allows developers to retrieve data from the JSONPlaceholder API using the `jsonf` command.

## Installation

Make sure you have Node.js installed on your machine. Then, install the JSON Fetch CLI globally using npm:

```bash
npm install -g json-fetch-cli
```
## Usage
After installing the CLI tool, you can use the jsonf command to interact with the JSONPlaceholder API. The available commands are:
  # Posts
  ```bash
  jsonf posts [postId] -p
  ```
  # Comments
  ```bash
  jsonf comments -p
  ```
  # Users
  ```bash
  jsonf users [userId] -p
  ```
## Options
-p, --pretty: Pretty-print the output from the API.
